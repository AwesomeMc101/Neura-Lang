# Programming
### Literals
* Two types, NUMBER and STRING. Define a string with either " or '

### Variables
* Define a variable with a type, like
```int foo = 5```
```string bar = "Hello"```
```const int foo = 7```

### Print
* Only function encoded atm, call with print("Hello World!") or print(bar).
